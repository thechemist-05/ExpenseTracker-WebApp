# Expense Tracker Web App

A full-stack Expense Tracker built using Flask and SQLite.

## Features

- Add, Edit, Delete Expenses
- Category-wise spending analysis
- Monthly expense filtering
- Daily spending trend visualization
- Dark / Light mode
- Responsive dashboard UI

## Tech Stack

- Python
- Flask
- SQLite
- Chart.js
- HTML/CSS

## Run Locally

```bash
pip install flask
python app.py
